# PyT - Python Taint

Static analysis of Python web applications based on theoretical foundations (Control flow graphs, fixed point, dataflow analysis)

Features planned:
- Detect Command injection
- Detect SQL injection
- Detect XSS
- Detect directory traversal

Work in progress
